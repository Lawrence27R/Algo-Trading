# 📈 Dhan Algo Trading Platform

AI-powered algorithmic trading platform using the [Dhan API](https://dhanhq.co/).
Supports **paper trading** (simulation) and **live trading** with an **interactive web dashboard**.

## 🚀 Features
- 🤖 AI/ML-based trading strategies (Random Forest)
- 📊 Technical indicator strategies (Moving Average Crossover, RSI)
- 📝 Paper trading simulator with virtual P&L tracking
- 💸 Live trading via Dhan API
- 🔁 Backtesting engine
- 🌐 Interactive Web Dashboard — track trades, visualize P&L, verify orders
- 📓 Jupyter notebooks for strategy research

## 🛠️ Setup
```bash
git clone https://github.com/Lawrence27R/Algo-Trading
cd Algo-Trading
pip install -r requirements.txt
cp .env.example .env
```

## ▶️ Run the Dashboard
```bash
cd dashboard
python app.py
```
Open: http://localhost:5000

## ⚠️ Disclaimer
Trading involves significant risk. Always test in paper mode first. This is for educational purposes.
